# vim setting
개인 Vim 셋팅입니다.

#### Install
```
cd ~ && git clone https://github.com/khw7096/vimset.git && cd vimset && sh install.sh && cd ~ && rm -rf ~/vimset
```

#### Hotkey
- F8 : run python
- F9 : run golang
