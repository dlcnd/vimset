# vim setting
* 개인 Vim 셋팅입니다.

##### install
```
cd ~ && git clone https://github.com/khw7096/vimset.git && sh ~/vimset/install.sh
go get golang.org/x/tools/cmd/goimports
```

#### Hotkey
- F5 : //주석 추가.(golang)
- F6 : //주석 제거.(golang)
- F8 : run python
- F9 : run golang
